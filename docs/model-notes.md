# Model notes

## What tends to be realistic on low VRAM

### More realistic
- 3B to 7B class models
- coding-focused instruct models
- quantized variants

### Less realistic
- large context expectations on weak GPUs
- smooth agentic workflows with bigger models
- "cloud-like" performance on consumer hardware

## Main selection criteria

When evaluating a model for low-end hardware, I care about:
1. fit in memory
2. usable response speed
3. coding quality
4. stability over longer sessions
5. ease of setup

## Important note

Many local AI discussions focus on what is technically possible.
This repo focuses more on what is **practically usable**.
